# secure-cli-password-manager
Uso de criptografia forte ( AES-256 através da biblioteca cryptography do Python) para salvar as outras senhas em um arquivo local modificado, ou seja, embora o arquivo json seja vazado, não seja possível descobrir a senha do usuário por meio do Hash. Além disso, o intuito do programa é descriptografar os dados se a senha mestre estiver correta.
